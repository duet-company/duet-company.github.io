DeepSeek

DeepSeek V4: 1 Trillion Parameter Architecture

Launched around March 3, DeepSeek V4 features 1 trillion parameters with MODEL1 architecture. Key innovations include tiered KV cache storage (cutting memory by 40%), Sparse FP8 decoding (1.8x inference speedup), and Engram for 1M+ token contexts.

  • 💾 40% memory reduction with tiered KV cache
  • ⚡ 1.8x inference speedup with Sparse FP8
  • 📚 30% pre-training efficiency improvement

Moonshot AI

Attention Residuals (AttnRes): Memory Optimization

Presented March 16, AttnRes enables transformer layers to attend to earlier layers via softmax attention and learnable queries. Reduces memory from O(Ld) to O(Nd), with Block AttnRes for grouped layers.

  • 🧠 Memory: O(Ld) → O(Nd)
  • 🔗 Earlier layer access via learnable queries
  • 📦 Block AttnRes for grouped layer optimization
