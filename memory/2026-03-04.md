**Critical Alerts:**

- Database pool exhausted (>90% used)
- High database error rate (>1/sec)
- Low disk space (<10% free)

- High API error rate (>0.1/sec)
- High API latency (P95 > 2s)
- Agent queue backlog (>100 tasks)

- High CPU usage (>80%)

- High query error rate
- Task queue backlog/failure rate
