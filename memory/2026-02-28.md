**2. Backup & Restore Automation Script**

**File:** `scripts/dev-tools/backup-automation.sh` (executable, 18KB)

**Features:**
- **Automated backups:**
  - **ClickHouse database** - Full SQL export with all tables
  - **PostgreSQL database** - pg_dump with schema and data
  - **Terraform state** - Backup all .tfstate files
  - **GitHub repositories** - Backup all git repos and workspace
  - **Workspace configuration** - MEMORY.md, SOUL.md, scripts, plans

- **Backup operations:**
  - `backup-all` - Backup all data sources
  - `backup-clickhouse` - Backup ClickHouse only
  - `backup-postgres` - Backup PostgreSQL only
  - `backup-terraform` - Backup Terraform state
  - `backup-github` - Backup GitHub repos
  - `backup-workspace` - Backup workspace config

- **Restore operations:**
  - `restore-clickhouse <file>` - Restore ClickHouse from backup
  - `restore-postgres <file>` - Restore PostgreSQL from backup

- **Backup management:**
  - `cleanup` - Remove old backups (older than BACKUP_RETENTION_DAYS)
  - `list` - List all backup files with sizes
  - `verify <file>` - Verify backup file integrity

- **Scheduling:**
  - `schedule [cron]` - Install automatic backup cron job
  - Default: Daily backups at 2 AM, weekly cleanup at 3 AM Sunday
  - Customizable schedule (e.g., every 6 hours)

- **Backup features:**
  - Compression (gzip) for all backups
  - Integrity verification (gzip and tar tests)
  - Timestamp-based filenames
  - Retention policy (default: 30 days)
  - Comprehensive logging to backup.log

- **Service health checks:**
  - Verifies Docker is running
  - Checks service health before backup
  - Waits up to 30 seconds for services to be ready
  - Reports errors and exits on failure

**Environment Variables:**
- `BACKUP_ROOT` - Backup directory (default: $WORKSPACE_ROOT/backups)
- `BACKUP_RETENTION_DAYS` - Retention period (default: 30)

**Usage Examples:**
```bash
# Full backup
./backup-automation.sh backup-all

# Backup specific database
./backup-automation.sh backup-clickhouse

# Restore from backup
./backup-automation.sh restore-clickhouse clickhouse_backup_20260228_120000.sql.gz

# Schedule automatic backups (every 6 hours)
./backup-automation.sh schedule "0 */6 * * *"

# List all backups
./backup-automation.sh list

# Verify backup integrity
./backup-automation.sh verify clickhouse_backup_20260228_120000.sql.gz
```

**Backup Directory Structure:**
```
backups/
├── clickhouse/              # ClickHouse backups
├── postgres/                # PostgreSQL backups
├── terraform-state/          # Terraform state backups
├── github/                   # GitHub repo backups
├── workspace/                # Workspace config backups
└── backup.log                # Backup operation logs
```
